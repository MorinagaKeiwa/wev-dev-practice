HTML

ページの構造を作るマークアップ言語。タグで要素に意味を付けます。

CSS

ページの見た目を整えるスタイルシート言語。色・フォント・レイアウトを担当。

JavaScript

ページに動きを付けるプログラミング言語。ボタン操作やアニメーションに使用。

Node.js

サーバーサイドで JavaScript を動かす環境。API やバックエンドの構築に使います。

Git

ソースコードのバージョンを管理するツール。変更履歴を記録し、チーム開発を支えます。

Vite

高速な開発用ビルドツール。React・Vue などのプロジェクトを素早く立ち上げられます。
