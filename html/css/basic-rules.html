Web 技術の基礎

HTML とは

HTML はページの構造を作るマークアップ言語です。タグを使ってコンテンツに意味を付けます。

CSS とは

CSS はページの見た目を整えるスタイルシート言語です。色・フォント・レイアウトを指定します。

JavaScript とは

JavaScript はページに動きを付けるプログラミング言語です。ボタン操作やアニメーションに使います。

優先順位の確認

① タグセレクタのみ（グレーになるはず）

② クラスセレクタあり（青になるはず）

③ ID セレクタあり（オレンジになるはず）
