Box Model の練習

Content（テキスト）がここに入ります

Padding の比較（背景色が広がる範囲に注目）

padding: 4px（窮屈）
padding: 16px（適切）
padding: 48px（広い）

Margin の比較（要素間の距離に注目）

margin-bottom: 4px
← 上の要素との距離が 4px
margin-bottom: 48px
← 上の要素との距離が 48px

Padding の 4 値指定（上 / 右 / 下 / 左）

padding: 40px(上) 8px(右) 8px(下) 8px(左)
上だけ広く空いているはず

Border のバリエーション

solid（実線）

最もよく使う枠線スタイルです。

dashed（破線）

セクションの仮枠などによく使います。

border-top でアクセントライン

上部だけに太い色付きの線を入れるデザインです。

box-sizing の比較（どちらも width: 300px）

実際の横幅の違いに注目してください。

content-box（デフォルト）
width: 300px + padding: 20px × 2 + border: 2px × 2
→ 実際の幅: 344px
border-box（推奨）
width: 300px の中に padding と border が含まれる
→ 実際の幅: 300px
