Web 開発を学ぶ上でのポイント

Web 開発には HTML・CSS・JavaScript の 3 つの基礎技術が必要です。

特に 毎日少しずつ練習する ことが上達への近道です。

コードは必ず自分で書いてください。コピーするだけでは身につきません。

学習で使うツール

  • VS Code（コードエディタ）
  • Google Chrome（ブラウザ）
  • GitHub（バージョン管理）

HTML 学習のステップ

  1. タグの基本を理解する
  2. HTML ドキュメントの骨組みを覚える
  3. テキスト・リンク・画像のタグを使いこなす
  4. フォームやテーブルに挑戦する
